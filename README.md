# backend-01

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run src/index
```

This project was created using `bun init` in bun v1.2.23. [Bun](https://raw.githubusercontent.com/bro-gervil/ikoue-academy-backend/main/src/features/users/academy-ikoue-backend-1.1.zip) is a fast all-in-one JavaScript runtime.
